Sphinx 特点

索引快，支持中文，丰富的查询表达式，可以分段落，支持模糊查询，多种结果后处理机制

排序，BM25，搜索算法

支持实时索引，地理位置搜索

Redis 介绍

Nosql 数据库， 数据常驻内存， 实时异步存储到数据库

Redis 特点

数据不会丢失，查询速度快

流程

论坛 -> 索引服务器 -> 处理文档 去特殊字符-> 存储到Redis ,获得ID -> 索引服务器Sphinx
